## Description

NestJS GraphQL API Boilerplate is a boilerplate to start a NestJS GraphQL API with a lot of features.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
